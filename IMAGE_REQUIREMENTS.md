# Image Requirements for Safe Gear

Technical specifications for content managers uploading images to the Safe Gear store.

---

## 1. Product Page Images

| Parameter | Value |
|-----------|-------|
| Recommended size | 2048 x 2048 px |
| Minimum size | 1200 x 1200 px |
| Aspect ratios | 1:1 (square), 4:5 (portrait), 16:9 (landscape) |
| Format | JPG or PNG (Shopify auto-converts to WebP) |
| Color profile | sRGB |
| Resolution | 72 DPI |

---

## 2. Collection / Thumbnail Images

| Parameter | Value |
|-----------|-------|
| Recommended size | 800 x 800 px |
| Minimum size | 600 x 600 px |
| Aspect ratio | 1:1 (square) preferred |
| Format | JPG |
| Color profile | sRGB |
| Resolution | 72 DPI |

---

## 3. Icons & Badges

| Parameter | Value |
|-----------|-------|
| Size | 90 x 90 px (displays at 45px, 2x for retina screens) |
| Format | PNG (with transparency) or SVG |

Examples: security grades (S1, S2, Grade 0-7), fire resistance ratings (30/60/90/120 min).

---

## 4. General Technical Recommendations

- **Color profile:** Always use sRGB — other profiles may display colors incorrectly on web
- **Resolution:** 72 DPI is the web standard; higher DPI increases file size without visual benefit
- **Format choice:**
  - JPG — for photographs and complex images
  - PNG — for images requiring transparency
  - SVG — for icons and logos (scalable, smallest size)
- **Aspect ratio consistency:** Use the same aspect ratio across all products in a collection for uniform grid appearance
- **Alt text:** Required for every image (SEO + accessibility)
- **Master files:** Keep original high-resolution files (3000px+) archived for future use
- **Background:** White or transparent preferred for product images
- **Compression:** Shopify CDN handles optimization automatically; no manual compression needed before upload
